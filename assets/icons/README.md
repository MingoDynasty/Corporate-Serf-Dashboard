# Local icons

These SVGs are the small set of icons used by the dashboard UI. They replace
runtime Iconify API requests so the local app keeps its chrome intact offline.

The files were copied from Iconify icon records for these collections:

- `bi`: Bootstrap Icons
- `clarity`: Clarity Icons
- `fontisto`: Fontisto
- `ion`: Ionicons
- `logos`: SVG Logos
- `material-symbols`: Material Symbols
- `radix-icons`: Radix Icons

When adding icons, vendor only the specific SVGs the app uses and confirm the
source collection's license before committing.
